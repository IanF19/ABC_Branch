Título: Información sobre Python

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Fue creado por Guido van Rossum y lanzado por primera vez en 1991.

Características principales:
- Sintaxis simple y fácil de aprender.
- Gran biblioteca estándar que permite trabajar con distintos tipos de datos y realizar muchas tareas sin necesidad de instalar paquetes adicionales.
- Compatible con varios paradigmas de programación: imperativo, orientado a objetos y funcional.
- Uso en diversas aplicaciones como desarrollo web, análisis de datos, inteligencia artificial, automatización, y más.

Python es conocido por su comunidad activa y su constante evolución. Algunas de las bibliotecas más populares son:
- NumPy: para operaciones matemáticas y científicas.
- Pandas: para análisis de datos.
- Django: para desarrollo web.

La filosofía de Python hace énfasis en la legibilidad del código y en la simplicidad, lo que lo convierte en una excelente opción tanto para principiantes como para programadores experimentados.
